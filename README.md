 ### 1.系统功能介绍

用户登录
> **1.管理员** <br>
 >>[1] 用户管理(目前未做，在数据库设置,后面会及时更新) <br>       
 >>>1.查看用户、修改用户、删除用户 <br> 
 >>>2、添加用户（可以赋予权限【0：管理员，1：学员】）<br> 
 
 
 >> [2] 课程管理<br> 
 >>>1、添加课程（课程编号、课程名称、授课教师、选课人数（默认为0）） <br>
 >>>2、查看已添加课程<br>
 >>>3、删除已添加课程<br>
  
> **2.学员** <br>
 >>[1] 我的选课 <br>
  >>>1、未选课程（学员选课）<br>
  >>>2、已选课程<br>
  
  ### 2.开发环境
