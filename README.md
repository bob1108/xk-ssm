# xk-ssm
first
基于ssm框架的选课系统
基本功能:
1、用户登录： 
  1> 管理员 
    [1] 用户管理(目前未做，在数据库设置) 
       1、查看用户、修改用户、删除用户 
       2、添加用户（可以赋予权限【0：管理员，1：学员】） 
   [2] 课程管理 
       1、添加课程（课程编号、课程名称、授课教师、选课人数（默认为0）） 
       2、查看已添加课程 
  2> 学员 
     [1] 我的选课
     1、未选课程（学员选课） 
     2、已选课程
开发语言：Java 
开发环境：win10+jdk1.8+IDEA+tomcat8.5+MySql5.5
